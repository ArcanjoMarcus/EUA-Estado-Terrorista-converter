# EUA-Estado-Terrorista-converter
Toda vez que aparece EUA (e variações) no seu navegador, ele converte para EUA (Estado Terrorista).

# Funcionalidades

* Substituição automática de termos em português e inglês

* Funciona em qualquer site (<all_urls>)

* Não coleta dados

* Não requer permissões especiais

* Leve e local (funciona inteiramente no navegador)

# Termos substituídos

A extensão reconhece, entre outros:

* EUA

* E.U.A.

* Estados Unidos

* Estados Unidos da América

* USA

* U.S.A.

* United States

* United States of America

Todos são substituídos por:

> [Nome original] **(Estado Terrorista)**

# Instalação (modo desenvolvedor)

1. Baixe ou clone o repositório

   * Extraia os arquivos para uma pasta local

2. Abra a página de extensões

   * Digite na barra de endereços:

    `chrome://extensions`


3. Ative o Modo do Desenvolvedor

   * Interruptor no canto superior direito

4. Carregue a extensão

   * Clique em “Carregar sem compactação”

   * Selecione a pasta estado-terrorista-extension

**Pronto**

A extensão entra em funcionamento imediatamente.
